Welcome to SquirrelUno's Documentation!
========================================

Welcome to the official documentation for SquirrelUno, an innovative project developed by Dominik Krenn that brings a unique twist to the classic Uno card game. This comprehensive guide provides all the necessary information to understand, install, and effectively utilize SquirrelUno.

.. warning::

   **Important Notice: SquirrelUno is an independent hobby project. The software may have stability and security issues and is in continuous development. Use at your own risk.**

Overview
========

SquirrelUno is designed to offer a fun and engaging card game experience with unique features and enhancements over the traditional Uno game. Whether you are a developer looking to contribute or a player eager to enjoy a new version of Uno, this documentation will guide you through every step.
